# EgzamIQ

### 🎯 Ace INF.04. Stop guessing. Start training smart.

[![MIT License](https://img.shields.io/badge/license-MIT-green?style=flat-square)](https://choosealicense.com/licenses/mit/)
[![React](https://img.shields.io/badge/React-18-61DAFB?style=flat-square&logo=react)](https://react.dev/)
[![Vite](https://img.shields.io/badge/Vite-5-646CFF?style=flat-square&logo=vite)](https://vitejs.dev/)
[![Status](https://img.shields.io/badge/status-active-brightgreen?style=flat-square)]()

**EgzamIQ** is an exam prep platform for the Polish INF.04 vocational certification that teaches you *why* — not just *what*.  
AI explains every answer. A smart planner tells you how many questions to tackle daily. Progress charts show whether you're actually improving.

[🚀 Demo](https://github.com/DragonFireEx/ExamIQ) · [📖 Docs](https://github.com/DragonFireEx/ExamIQ/blob/main/docs/examiq_dokumentacja.html) · [🐛 Report a bug](https://github.com/DragonFireEx/ExamIQ/issues)

## ✨ Why EgzamIQ?

| Feature | What it does for you |
|---|---|
| 🤖 **AI Explanations** | Every answer — right or wrong — gets an instant breakdown |
| 📊 **Daily Progress Chart** | See your learning trend, not just a single test score |
| 🗓️ **Exam Planner** | Set your exam date, get a daily question target tailored to your goal |
| ⚡ **Instant Feedback** | No waiting — answer, get your result, move on |

## 🚀 Running Locally

```bash
# Clone the repo
git clone https://github.com/DragonFireEx/egzamiq.git
cd egzamiq

# Install dependencies
npm install

# Start the dev server
npm run dev
```

App will be live at `http://localhost:5173`
