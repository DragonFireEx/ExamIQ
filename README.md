# ExamIQ

**EgzamIQ** is a web application designed to help users prepare for Polish technical  exams. It offers **AI-powered** multiple choice tests with instant explanations for every answer, a **daily progress chart** to visualize learning trends over time, and a smart **study planner** that calculates how many questions you need to practice each day to reach your target score by exam date.
